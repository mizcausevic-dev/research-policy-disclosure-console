# Security Policy

This repository is a public demonstration surface using **synthetic research disclosure data only**.

## Scope

- No live student, faculty, sponsor, committee, or production research records should be committed here.
- The repo demonstrates research-policy routing and evidence-posture patterns, not a production university compliance platform.

## Disclosure

If you identify a security issue in the code or deployment workflow, please open a private report through GitHub Security Advisories if available, or contact the repository owner directly.

## Operational note

This project is **not** represented as FERPA-, HIPAA-, IRB-, or grant-audit compliant by default. Any production adaptation would need formal review, validation, data-handling controls, and environment-specific hardening.
